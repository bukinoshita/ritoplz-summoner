## Install

```bash
$ npm install --save ritoplz-summoner
```

## Usage

```js
const summonerCover = require('ritoplz-summoner')

summonerCover
// => https://ritoplz-summoner-cover.now.sh/leona.png
```

## API

### summonerCover

Return a random url
